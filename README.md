# Movie Search Engine

This is a simple movie search engine where users can search for movies and add them to their favourites.
The project was built using JavaScript, HTML, and CSS with React, integrates the TMDB API for movie data, and is deployed on Vercel.

## Website

[Check it out here](https://movie-search-engine-cyan.vercel.app/)

## Features

- Search for movies using the TMDB API
- Serverless API routes
- Browse popular movies
- Add and remove movies from favourites
- Favourites persist using localStorage
- Responsive movie grid layout

## Tech Stack

- React
- JavaScript
- HTML
- CSS
- TMDB API
- Vercel
